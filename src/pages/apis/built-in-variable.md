# Built-in Variables

| Variables | Description | Type |
| ----------|-------------| -----|
| `sharedA` ... `sharedE` | Variable that is global amongst all your bots. | `undefined` |
| `life` | The Bot's current life. | `integer` |
| `x` | The Bot's coordinate on the X-Axis. | `integer` |
| `y` | The Bot's coordinate on the Y-Axis. | `integer` |
| `arenaWidth` | The width of the Arena. | `integer` |
| `arenaHeight` | The height of the Arena. | `integer` |
